    Elementos HTML embutidos

    HTML define uma longa lista de tags embutidas disponíveis, uma lista completa das quais pode ser encontrada na Mozilla Developer Network.

    • Para texto em negrito, use <strong>.
    • Para texto em itálico, use <em>.
    • Abreviações, como HTML devem usar <abbr>, com um title para a frase completa.
    • Citações, como — Mark Otto, devem usar <cite>.
    • Texto excluído deve usar <del> e texto inserido deve usar <ins>.
    • Sobrescrito text usa <sup> e subscrito text usa <sub>.

    A maioria desses elementos são estilizados por navegadores com poucas modificações de nossa parte.
